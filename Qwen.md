# PPBJ Project Notes

This file contains notes and context about the PPBJ (Pusat Pembelajaran Bahasa Jepang) project, part of Kerja Sama Sekolah Vokasi IPB.

## Project Overview
- Project Name: Komisi Kerja sama Sekolah Vokasi IPB - PPBJ (Pusat Pembelajaran Bahasa Jepang)
- Framework: Next.js (App Router)
- Styling: Tailwind CSS
- Purpose: Website company profile for the cooperation commission of IPB Vocational School, featuring details about various programs including the Japanese Language Learning Center (PPBJ)

## Recent Changes
- Completely redesigned the homepage with modern styling
- Implemented responsive design with improved layout
- Added animated background elements using CSS animations
- Created attractive cards for navigation (About Us, Programs, Contact)
- Added icons and hover effects to improve user experience
- Improved typography with gradient text effects
- Fixed HTML structure by creating proper root layout with html/body tags
- Added custom animations for blob effects in background

## File Structure
- `/app/page.tsx` - Main landing page with new design
- `/app/layout.tsx` - Root layout with html/body tags
- `/app/globals.css` - Global styles and custom animations
- `/Qwen.md` - Project notes and context

## Technical Implementation
- Used Tailwind CSS utility classes for styling
- Implemented responsive design with grid and flexbox
